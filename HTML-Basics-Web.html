HTML Basics

HTML stands for "hypertext markup language" and is a markup language used to represent structure data. Writing HTML doesn't need to be difficult, especially if you keep these ideas in mind.

HTML is a specification

HTML is a specification, or set of recommendations for how it should be written. It is maintained by the W3C and you can read the specification here.

HTML represents content, not style

A web document has two primary components: content, which determines what to render on the screen, and style, which determines how to render that content. HTML is the part that represents the content of a web document; CSS represents the style.

HTML is made up of elements

HTML is constructed from building blocks called elements, which consist of an opening tag that indicates the start of an element, a closing tag that indicates the end of an element, and content in between the opening and closing tags. Only the content is made visible when the page is rendered on the screen. For example, <p>This is a paragraph</p> represents a paragraph element.

An HTML document consists of a few key elements:

  1. A doctype declaration (<!DOCTYPE html>)
  2. an <html> element
  3. a <head> element
  4. a <body> element

HTML is block-directed

HTML elements are block-directed, meaning they behave as though they are contained within imaginary rectangles that cause them to overflow to the next line in the page, forcing a line break. Some elements are inline elements, meaning they don't cause a line break to occur.

HTML structure is nested

HTML elements can be nested within other elements. This means the "rectangles" around HTML elements can be encapsulated within other rectangles. If Element B is nested inside Element A, we say that Element B is a child element of Element A; conversely, Element A is the parent element of Element B.

HTML is semantic

HTML elements represent parts of a structured document that have specific meaning. Every tag name in HTML has a meaning that should match the type of content it is used to represent in markup. For example, the <p> tag is used to represent paragraph text.

Here are some common HTML tags and their meanings:

  • h1 - h6: heading text
  • p: paragraph text
  • ul: an unordered (bulleted) list
  • ol: an ordered (numbered) list
  • li: an item in a list
  • b: text that should be made bold and visually distinct from other text
  • i: text that should be made italic and visually distinct from other text
  • strong: text that is vocally stressed
  • em: text that is vocally emphatic
  • div: a section or division of thematically related content
